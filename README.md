Archivos disponibles

1. ejercicio1.py → contiene la función bigSorting.
2. ejercicio2.py → contiene la función camelcase.

Cómo ejecutar cada programa
1. Abre una terminal o consola en la carpeta donde guardaste los archivos.

2. Para correr el programa con bigSorting escribe:

    python ejercicio1.py

3. Esto mostrará en pantalla el resultado de la búsqueda del número en el arreglo.

4. Para correr el programa con camelcase escribe:

   python ejercicio2.py
5. Esto mostrará en pantalla el número de palabras detectadas en la cadena.
